# LinkedIn Job Buddy
End-to-end ML project: Predict job salary & recommend similar jobs from LinkedIn data.

## Setup
pip install -r requirements.txt

## Run Notebook
jupyter notebook notebooks/main.ipynb  # Train & save models

## Run App
streamlit run app.py

## Large Files Notice
Datasets and trained models are excluded due to GitHub size limits.

To run the project:
1. Download datasets manually
2. Place them in: job-buddy/data/
3. Place trained models in: job-buddy/src/models/
